### 在线阅读
> **[omega.dw.cash](https://omega.dw.cash/)**

本项目使用 [mdBook](https://github.com/rust-lang/mdBook) 构建。

### 前置要求
- 安装 Rust 和 Cargo。
- 安装 mdBook：
  ```bash
  cargo install mdbook
  ```

### 构建书籍

1.  克隆仓库：
    ```bash
    git clone https://github.com/loning/the-omega.git
    ```
2.  进入目录：
    ```bash
    cd the-omega
    ```
3.  本地服务（热重载）：
    ```bash
    mdbook serve --open
    ```
    书籍将在 `http://localhost:3000` 可用。
